# Roadmap — da demo à produção

Este documento descreve a evolução comercial e técnica **depois** da demo apresentada à cliente.

## Estado actual (demo)

- Site estático em produção na Vercel
- PT/EN, WhatsApp, formulário (modo demo → WhatsApp)
- SEO técnico e performance tratados
- Identidade e contactos reais do gabinete

## Fase 0 — Fecho comercial

- [ ] Validação da cliente (copy, foto, serviços)
- [ ] Acordo de entrega (one-shot + opcional manutenção)
- [ ] Domínio escolhido (ex.: `lnunes-contabilista.pt`)

## Fase 1 — Produção oficial (1 semana)

- [ ] Domínio + HTTPS na Vercel
- [ ] Actualizar `config.js`, canonicals, sitemap, robots
- [ ] Formspree (ou equivalente) no formulário
- [ ] Google Search Console + sitemap
- [ ] Google Business Profile alinhado (NAP)

## Fase 2 — Conversão (2 semanas)

- [ ] Página / estado de “pedido enviado”
- [ ] Reforço sazonal IRS (já existe bloco com ano automático)
- [ ] GA4 com consentimento (já preparado em `analytics.js`)

## Fase 3 — Confiança (2–4 semanas)

- [ ] 3–5 testemunhos reais (com autorização escrita)
- [ ] Secção “como começamos a trabalhar”
- [ ] FAQ expandida com dúvidas reais da cliente

## Fase 4 — Operações (quando houver volume)

- [ ] Integração CRM / Teglion
- [ ] Agendamento (Calendly ou similar)
- [ ] WhatsApp Business API (opcional)

## Fase 5 — Produto (freelance)

- [ ] Extrair template “gabinete contabilístico”
- [ ] `config.js` + tokens CSS por cliente
- [ ] Checklist de onboarding reutilizável
